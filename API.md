## Crsl
**Kind**: global class  
<a name="new_Crsl_new"></a>

### new Crsl(selector, [options])
A lightweight, customizable, extendable, responsive image slider in vanilla JS


| Param | Type | Default | Description |
| --- | --- | --- | --- |
| selector | <code>string</code> |  | CSS selector for the root carousel element. |
| [options] | <code>object</code> |  |  |
| [options.enableDots] | <code>boolean</code> | <code>false</code> | Whether to enable dot controls. |
| [options.duration] | <code>number</code> | <code>800</code> | Transition duration in ms. |
| [options.autoplay] | <code>boolean</code> | <code>false</code> | Enable autoplay. |
| [options.interval] | <code>number</code> | <code>2000</code> | Autoplay interval in ms. |
